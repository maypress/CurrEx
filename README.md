# CurEx 💱 Central Bank of the Russian Federation Currency Converter

[![Go](https://img.shields.io/badge/Go-1.22%2B-blue.svg)](https://golang.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**Fast CLI + TUI currency converter based on the official exchange rates of the Central Bank of the Russian Federation.**

Retrieves data from the [Central Bank of the Russian Federation](https://www.cbr.ru/scripts/XML_daily.asp) via the mirror `cbr-xml-daily.ru`. 50+ currencies, caching, CSV export.

## ✨ Quick Start

```bash
# Clone and run
git clone https://github.com/maypress/CurEx.git
cd CurEx
go mod tidy
```

## 🚀 Usage

### ✅ **CLI with flags (basic syntax)**
```bash
# Convert USD → EUR
PS C:\CurrEx> go run cmd/currex/main.go --amount 100 --from USD --to EUR
💱 100.00 USD = 92.50 EUR

# Other examples
go run cmd/currex/main.go --amount 50 --from RUB --to USD
go run cmd/currex/main.go --amount 1000 --from EUR --to RUB
```

### 📱 **Graphical Interface**
```bash
go run cmd/currex/main.go --ui
```

### 📋 **Useful Commands**
```bash
go run cmd/currex/main.go --list # List all currencies
go run cmd/currex/main.go --csv # Export exchange rates to CSV
```

## 🌐 **API Source**

**XML Central Bank of the Russian Federation:** `https://www.cbr-xml-daily.ru/daily_utf8.xml` [web:6]

```xml
<Valute>
<CharCode>USD</CharCode>
<Nominal>1</Nominal>
<Value>92.50</Value> <!-- for 1 USD -->
</Valute>
```

## 🐛 **FAQ**

| ❌ Problem | ✅ Solution |
|-------------|------------|
| `go: command-line arguments` | `go run cmd/currex/main.go --amount 100 --from USD --to RUB` |
| `TUI is buggy` | `export TERM=xterm-256color` |
| `Rates = 0` | `rm -rf cache/; go run ...` |
| `403 Forbidden` | CBR mirror already in use |
